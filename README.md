## ✨ Overview

An interactive **Simon Game web application** where players must **remember and repeat an increasing sequence of colors**.

This project demonstrates **game logic, event handling, animations, and user interaction** using JavaScript.

---

## 🚀 Features

* 🎨 Color sequence generation
* 🧠 Memory-based gameplay
* 🔊 Sound effects for each button
* ⚡ Increasing difficulty with levels
* ❌ Game over detection
* 🔄 Restart functionality

---

| Technology | Purpose                  |

| HTML       | Structure                |

## 🎮 How to Play

```bash
1. Press any key to start the game
2. Watch the color sequence carefully
3. Repeat the same sequence by clicking buttons
4. Each level adds a new step
5. Make a mistake → Game Over ❌
```
